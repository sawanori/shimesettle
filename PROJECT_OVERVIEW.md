# プロジェクト概要: NonTurn決算申告

## 1. プロジェクトの目的
このプロジェクトは、経費と売上を管理するためのWebアプリケーション「NonTurn決算申告」の開発です。
主なターゲットはフリーランスや小規模事業者と想定され、領収書のAI解析（OCR）による経費入力の自動化、売上管理、そして経営状況の可視化（ダッシュボード）を特徴としています。また、税理士向けのデータ出力機能も計画されています。

## 2. 技術スタック
- **フレームワーク**: Next.js 16 (App Router)
- **言語**: TypeScript
- **スタイリング**: Tailwind CSS 4
- **UIコンポーネント**: shadcn/ui (導入予定)
- **バックエンド**: Supabase (Database, Auth, Storage)
- **AI機能**: OpenAI API (GPT-4o) を利用した領収書解析
